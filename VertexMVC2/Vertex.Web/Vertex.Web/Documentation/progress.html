Progress

Documentation and examples for using Bootstrap custom progress bars featuring
support for stacked bars, animated backgrounds, and text labels.

Default

Use @Html.Vertex().Progress() method to create a progress

  • Preview
  • ASP.NET MVC 




@Html.Vertex().Progress().Value(0)
@Html.Vertex().Progress().Value(25)
@Html.Vertex().Progress().Value(50)
@Html.Vertex().Progress().Value(75)
@Html.Vertex().Progress().Value(100)

Height

use .Height(string height) method to set height

  • Preview
  • ASP.NET MVC 



@Html.Vertex().Progress().Value(25).Height("1px")
@Html.Vertex().Progress().Value(25).Height("4px")
@Html.Vertex().Progress().Value(50).Height("8px")
@Html.Vertex().Progress().Value(75).Height("2.6rem")

Labels

use .Label(string label) method to set label

  • Preview
  • ASP.NET MVC
25%

Bootstrap 
@Html.Vertex().Progress().Value(25).Label("25%")
@Html.Vertex().Progress().Value(50).Label("Bootstrap")

Color

use .Color(BootstrapColor color) method to set color. Default color is BootstrapColor.Primary

  • Preview
  • ASP.NET MVC 





@Html.Vertex().Progress().Value(25).Color(BootstrapColor.Danger)
@Html.Vertex().Progress().Value(40).Color(BootstrapColor.Dark)
@Html.Vertex().Progress().Value(55).Color(BootstrapColor.Primary)
@Html.Vertex().Progress().Value(70).Color(BootstrapColor.Secondary)
@Html.Vertex().Progress().Value(85).Color(BootstrapColor.Success)
@Html.Vertex().Progress().Value(100).Color(BootstrapColor.Warning)

Striped

use .Striped(bool animatedStripes = false) method to set striped.

  • Preview
  • ASP.NET MVC

Animated Stripes

@Html.Vertex().Progress().Value(55).Striped()
@Html.Vertex().Progress().Value(85).Striped(true)

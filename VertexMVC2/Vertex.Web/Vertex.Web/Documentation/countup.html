Countup

The Count-up shortcode offers a dynamic display of numbers counting up from zero when a visitor initially views the page. This shortcode is a good way to promote stats or data in an engaging way.

Default

Use @Html.Vertex().Countup() method to create a countup

  • Preview
  • ASP.NET MVC 

@Html.Vertex().Countup().FromTo(0, 800)

Prefix

use .Prefix(string prefix) method to set prefix

  • Preview
  • ASP.NET MVC 

@Html.Vertex().Countup().FromTo(100, 1250).Prefix("$")

Suffix

use .Suffix(string suffix) method to set suffix

  • Preview
  • ASP.NET MVC 

@Html.Vertex().Countup().FromTo(500, 3000).Suffix("K")

Duration

use .Duration(int duration) method to set duration

  • Preview
  • ASP.NET MVC 

@(Html.Vertex().Countup()
        .FromTo(150, 2000)
        .Prefix("$")
        .Suffix("M")
        .Duration(1500))

Popovers

Documentation and examples for adding Bootstrap popovers, like those found in iOS, to any element on your site.

Default

Use @Html.Vertex().Popover() method to create a Popover. Use .For(...)(+3 overloads) method to bind any html element, html string or vertex component.

  • Preview
  • ASP.NET MVC
Popover example
Witout title 
@Html.Vertex().Popover()
    .Title("Hello world")
    .Content("popover content")
    .For(Html.Vertex().Button("Popover example"))
                                
@Html.Vertex().Popover()
     .Content("popover content")
     .For(@<text> <a href="#">Witout title</a> </text>)

use .For(...)(+3 overloads) method to bind any html element.

// For button 
@Html.Vertex().Popover()
    .Title("Hello button")
    .Content("popover content")
    .For(Html.Vertex().Button("Popover Button").Size(ButtonSize.Small))

// For countup 
@Html.Vertex().Popover()
    .Title("Hello countup")
    .Content("popover content")
    .Direction(PopoverDirection.Left)
    .For(Html.Vertex().Countup().FromTo(0, 850).Prefix("$"))

// For progress 
@Html.Vertex().Popover()
    .Title("Hello progress")
    .Content("popover content")
    .For(Html.Vertex().Progress().Value(100))

// For checkbox 
@Html.Vertex().Popover()
    .Title("Hello checkbox")
    .Content("popover content")
    .Direction(PopoverDirection.Left)
    .For(@<text>
            <div class="custom-control custom-checkbox">
              <input type="checkbox" class="custom-control-input" checked="">
              <label class="custom-control-label">Checked</label>
            </div>
          </text>)

// For form control 
@Html.Vertex().Popover()
    .Title("Hello form control")
    .Content("popover content")
    .For(@<text>
            <div class="form-group m-0">
                <input class="form-control" type="text" placeholder="First name">
            </div>
          </text>)

// For link
@Html.Vertex().Popover()
    .Title("Hello link")
    .Content("popover content")
    .For(@<text>
            <a href="#" class="m-link">Popover link</a>
          </text>)
